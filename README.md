## English

### ProperS | Audio Visualizer
High-performance, browser-based audio analysis subsystem. Designed for the real-time monitoring of spectral and spatial audio metrics.

#### Core Features
*   **Master Bus Metering:** Real-time extraction of RMS, Peak, LUFS (Integrated), and clipping detection.
*   **Spatial Metrics:** Integrated Vector Scope (Goniometer) for stereo imaging, phase correlation, and stereo widening analysis.
*   **Spectral Dynamics:** Calculation of Frequency Centroid and Crest Factor.
*   **Multi-Mode Visualization:** Includes FFT Spectrum, EQ Bars, Radial FFT, Waterfall, and Waveform Dat.
*   **System Integration:** Playlist queue support for multiple tracks, adjustable FFT buffer sizes (2048, 4096, 8192), and customizable UI tint hex.

#### Deployment
Zero dependencies. Execute the `html` file natively in any modern web browser.
