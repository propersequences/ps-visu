## 🇬🇧 English

### ProperS | Audio Visualizer

High-performance, browser-based audio analysis subsystem.  
Designed for real-time monitoring of spectral and spatial audio metrics.

#### ▸ Core Features

- **Master Bus Metering**  
  Real-time extraction of RMS · Peak · LUFS (Integrated) · Clipping detection

- **Spatial Metrics**  
  Integrated Vector Scope (Goniometer) → stereo imaging · phase correlation · stereo widening analysis

- **Spectral Dynamics**  
  Frequency Centroid · Crest Factor calculation

- **Multi-Mode Visualization**  
  FFT Spectrum · EQ Bars · Radial FFT · Waterfall · Waveform Data

- **System Integration**  
  Playlist queue · FFT buffer sizes (2048 / 4096 / 8192) · Custom UI tint (hex)

#### ▸ Deployment

Zero dependencies.  
Execute the `html` file natively in any modern browser.
