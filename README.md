## English

### SYS.V4_CORE
High-performance, browser-based audio analysis subsystem[cite: 3]. Designed for the real-time monitoring of spectral and spatial audio metrics.

#### Core Features
*   **Master Bus Metering:** Real-time extraction of RMS, Peak, LUFS (Integrated), and clipping detection[cite: 3].
*   **Spatial Metrics:** Integrated Vector Scope (Goniometer) for stereo imaging, phase correlation, and stereo widening analysis[cite: 3].
*   **Spectral Dynamics:** Calculation of Frequency Centroid and Crest Factor[cite: 3].
*   **Multi-Mode Visualization:** Includes FFT Spectrum, EQ Bars, Radial FFT, Waterfall, and Waveform Dat[cite: 3].
*   **System Integration:** Playlist queue support for multiple tracks, adjustable FFT buffer sizes (2048, 4096, 8192), and customizable UI tint hex[cite: 3].

#### Deployment
Zero dependencies. Execute the `html` file natively in any modern web browser.

## Español

### SYS.V4_CORE
Subsistema de análisis de audio de alto rendimiento ejecutado en navegador[cite: 3]. Diseñado para la monitorización en tiempo real de métricas de audio espectrales y espaciales.

#### Características Principales
*   **Medición de Bus Maestro:** Extracción en tiempo real de RMS, Peak, LUFS (Integrado) y detección de saturación (clipping)[cite: 3].
*   **Métricas Espaciales:** Vector Scope (Goniómetro) integrado para imagen estéreo, correlación de fase y análisis de amplitud estéreo[cite: 3].
*   **Dinámicas Espectrales:** Cálculo de Centroide de Frecuencia y Factor de Cresta[cite: 3].
*   **Visualización Multimodelo:** Incluye Espectro FFT, Barras EQ, FFT Radial, Cascada (Waterfall) y Datos de Forma de Onda[cite: 3].
*   **Integración de Sistema:** Soporte de cola de reproducción para múltiples pistas, tamaños de búfer FFT ajustables (2048, 4096, 8192) y color de interfaz personalizable[cite: 3].

#### Despliegue
Cero dependencias. Ejecutar el archivo `html` de forma nativa en cualquier navegador web moderno.
